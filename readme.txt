= How it works =

== Installation ==

1. Upload `plugin-faq-parser` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

How to use

= Shortcode =

Please add new Page. Please write a short code.

Examples

[pluginfaq slug="media-from-ftp"]

[pluginfaq slug="media-from-ftp" bdback="#ffffff"]

* slug		Specifies the plugin slug.

Question
* bdline	Border Color
* bdback	Background Color
* bdtext	Text Color

Answer
* back		Background Color
* text		Text Color
